# JVM

## JVM 왜 쓸까?

Write Once Run Anywhere

## JVM 구성요소

- Class Loader
- Execution Engine(실행 엔진): 기계어로 바꾸어줌
- Garbage Collector
- Runtime Data Area
    - Method Area
    - Heap Area
    - Stack Area
    - PC register
    - Native Method Stack
